# Procurement Notes

These notes support enterprise review of WaForge materials.

## Classification

WaForge materials may include prompts, documents, source files, JSONL data, local AI import packages, runtime instructions, and release manifests.

## Review points

Enterprise reviewers should check:

- source terms
- commercial license status
- support terms
- privacy screen
- security screen
- provenance records
- dependency list
- deployment scope
- warranty position
- support contact
- maintenance expectations

## Commercial engagement

Commercial or enterprise deployment should use a separate order form, statement of work, support schedule, or commercial license.

## No assumed endorsement

No customer, sponsor, celebrity, family member, or public figure should be represented as endorsing WaForge unless written approval exists.
