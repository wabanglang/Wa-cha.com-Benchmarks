# WaForge Legal Foundry

This folder contains the WaForge legal and governance document set for ownership, source terms, contributor handling, commercial licensing, trademark use, provenance, release review, and project stewardship.

This material is a drafting pack for review by qualified counsel. It is not legal advice.

## Core principle

WaForge materials should remain traceable, attributable, reviewable, and stewarded by the project owner.
